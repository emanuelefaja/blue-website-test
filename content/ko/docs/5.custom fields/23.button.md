## 버튼 커스텀 필드 생성하기

프로젝트에서 다른 커스텀 필드를 생성하는 것과 동일한 방식으로 버튼 커스텀 필드를 생성할 수 있습니다.

![Create Button Custom Field](/product/button-field-create.png "Create Button Custom Field")

버튼에 이름을 지정하고 확인 유형을 선택할 수 있습니다.

## 확인

확인 기능은 사용자에게 팝업을 표시하여 버튼을 누를 것인지 확인하도록 함으로써 실수로 버튼을 클릭하는 것을 방지하도록 설계되었습니다.

![Button Confirmation](/product/button-confirmation.png "Button Confirmation")

확인에는 세 가지 옵션이 있습니다:

1. **확인 없음** — 버튼을 누르는 즉시 자동화 작업이 트리거됩니다.
2. **소프트 확인** — 확인 팝업이 표시되고, 사용자는 확인 버튼을 누르기만 하면 됩니다.
3. **하드 확인** — 확인 팝업이 표시되고, 사용자는 특정 문구를 입력한 다음 확인 버튼을 눌러야 합니다.

고객 확인 이메일 발송과 같은 중요한 작업을 트리거하는 자동화 작업과 연결된 버튼에는 하드 확인을 사용하는 것을 권장합니다.

## 버튼을 자동화 작업에 연결하기

특정 버튼을 Blue에서 지원하는 모든 자동화를 트리거할 수 있는 작업으로 선택할 수 있습니다.
