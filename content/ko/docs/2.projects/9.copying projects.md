프로젝트 설정 아이콘을 클릭하여 조직 내에서 또는 다른 조직으로 프로젝트를 복사할 수 있습니다. [프로젝트 관리자](/docs/user-management/roles/project-administrator)만 프로젝트를 복사할 수 있습니다.

![Copying Project 1](/docs/CopyingProject_1.png "Copying Project 1")

프로젝트 관리자만 프로젝트 설정에 액세스할 수 있습니다

프로젝트를 복사할 때 복사하고 싶은 내용을 선택할 수 있습니다.

![Copying Project 1](/docs/CopyingProject_2.png "Copying Project 2")

복사하고 싶은 프로젝트 구성 요소를 선택하세요
