활동 추적에는 세 가지 수준이 있습니다:

1. **조직 활동** - 회사 전체의 모든 프로젝트에서 발생하는 모든 일을 로깅합니다.
2. **프로젝트 활동** - 특정 프로젝트 내의 작업을 로깅합니다.
3. **레코드 활동** — 레코드의 세부적인 변경 로그를 제공합니다.

![Activity 1](/docs/Activity_1.png "Activity 1")

회사 및 프로젝트 활동

활동은 조직 전체의 활동에 대한 완전한 감사 추적을 제공합니다. 지속적인 현황 회의 없이도 회사 전반에서 무엇이 일어나고 있는지 이해할 수 있게 해줍니다.

### 타임라인 필터링

![Activity 2](/docs/Activity_2.gif "Activity 2")

담당자별 타임라인 필터링 예시

활동은 다음으로 필터링할 수 있습니다:

**사용자** 필터를 사용하면 선택된 한 명 이상의 사용자와 관련된 활동을 볼 수 있습니다. 사용자 필터에 포함되는 내용은 다음과 같습니다:

이 필터는 선택된 사용자와 연관된 활동을 표시합니다:

- 사용자에게 직접 할당된 레코드에서 수행된 활동
- 사용자가 작성한 새 댓글
- 사용자가 생성한 새 레코드
- 사용자가 사용자 정의 필드에 수행한 변경사항
- 사용자가 수행한 기타 수정 또는 업데이트
- 사용자가 언급된 경우

사용자 필터를 적용하면 특정 개인이 수행한 작업과 변경사항을 빠르게 분리하고 검토할 수 있어, 조직 내에서 그들의 기여도와 활동을 추적하기 쉬워집니다.

![Activity 3](/docs/Activity_3.png "Activity 3")

**태그** 이 필터는 하나 이상의 특정 태그로 태그된 레코드와 관련된 활동을 표시합니다.

특정 태그를 선택하면 해당 태그와 연관된 레코드에서 수행된 활동의 타임라인을 볼 수 있습니다. 이 필터는 태그로 식별되는 특정 프로젝트, 카테고리 또는 관심 영역과 관련된 활동에 집중하고자 할 때 특히 유용합니다.

![Activity 4](/docs/Activity_4.png "Activity 4")

**날짜** 이 필터를 사용하면 날짜 범위를 지정할 수 있으며, 선택된 기간 내에 발생한 활동을 표시합니다.

시작 날짜와 종료 날짜를 설정하여 특정 주, 월 또는 사용자 정의 날짜 범위와 같은 특정 기간 동안 발생한 활동에 집중하도록 타임라인을 좁힐 수 있습니다.

이 필터는 과거 활동을 검토하고, 정의된 타임라인에 따른 진행 상황을 추적하거나, 특정 시간 프레임 내의 활동을 분석할 때 유용합니다.

### 읽지 않음 강조 표시

읽지 않은 활동은 파란색으로 강조 표시되어 새로운 작업을 쉽게 식별할 수 있습니다. 클릭하면 해당 활동의 세부 정보가 표시됩니다.

편집할 수 없는 로그로서 회사 활동은 모든 사용자 작업의 신뢰할 수 있는 기록을 제공합니다. 모든 팀과 프로젝트 간의 인지도와 투명성을 촉진합니다.
