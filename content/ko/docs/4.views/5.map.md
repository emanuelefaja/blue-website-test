## 핀 & 추적

레코드에서:

- **이동 중 핀 찍기**: 위치 아이콘을 사용하여 현재 위치를 즉시 핀으로 찍을 수 있습니다.

- **위치 추적**: 지도는 [위치/지도 커스텀 필드](/docs/custom-fields/location-map)를 활용하여 레코드와 연결된 위도 및 경도 데이터를 가져옵니다.

- **시각적 식별**: 지도 핀은 레코드의 기본 태그를 기반으로 색상 코딩되어 직관적인 인식을 가능하게 합니다.

![Map 2](/docs/Map_2.png "Map 2")

지도에서:

- **세부 정보 팝업**: 핀을 클릭하면 레코드의 이름, 프로젝트, 목록, 태그, 담당자가 포함된 팝업이 표시됩니다. 다시 클릭하면 전체 레코드가 열립니다.

![Map 3](/docs/Map_3.png "Map 3")

- **탐색 컨트롤**: 확대, 이동, 현재 위치로 스냅이 가능합니다.

- **지속적인 필터**: 다른 뷰에서 적용된 모든 [필터](/docs/views/filters)가 지도에서도 유지됩니다.

![Map 4](/docs/Map_4.png "Map 4")

## Google Maps

Blue는 기본 OpenStreetMap 대신 지도 뷰에 Google Maps를 사용하는 것도 지원합니다. Google Maps는 특정 지역의 더 상세한 지도와 더 정확한 위치 데이터를 제공하는 경우가 많습니다. 그러나 이를 위해서는 Google Maps API 키가 필요하며, 회사 소유자가 회사 설정에서 입력해야 합니다.

**사용 사례**

지도 뷰는 다양한 사용 사례에서 독특한 관점을 제공합니다:

- 고객, 파트너 또는 자산 위치 표시
- 서비스 지역, 영업 구역 또는 시장 영역 시각화
- 배송 출발지 및 목적지 추적
- 이벤트 장소, 작업 현장 또는 재고 창고 매핑
- 운영 분석을 위한 시설 위치 식별
- 현장 직원 위치 모니터링
- 지리적 자산 및 관심 지점 표시

대화형 탐색과 내장된 위치 추적을 통해 지도 뷰는 데이터 내에서 새로운 차원을 드러냅니다.
