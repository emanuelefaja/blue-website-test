Det anpassade Procent-fältet möjliggör att fånga och beräkna procentvärden inom Blue-poster. Detta möjliggör framstegsspårning, prestationsmätning och datavisualisering baserad på procent.

![Percent](/docs/Percent.png "Percent")

Det anpassade Procent-fältet kommer med robust validering för att stödja noggrann data. Vid inmatning av värden kontrollerar det att numret är inom det standardmässiga 0-100% området som förväntas för procent. Detta förhindrar att ogiltig data introduceras.

En nyckelfunktion är den flexibla konfigurationen av decimalprecision. Du kan ställa in fältet för att fånga procent med så många decimaler som krävs baserat på dina behov. Denna nivå av anpassning möjliggör procentbaserad dataspårning på vilken granularitetsnivå som helst som ditt användningsfall kräver.

För att förenkla beräkningar integrerar Procent-fältet direkt med [Formula anpassade fält](/sv/docs/custom-fields/formula). Detta gör det enkelt att automatisera framstegsstaplar, förhållanden, priser och andra vanliga procentbaserade mätvärden. Beräkningarna sker bakom kulisserna och visar utdataprocentvärdet.

Vid export av data via [CSV](/sv/docs/integrations/csv-export) eller [API](/sv/docs/integrations/api) integrationer behåller fältet sin formaterade procentvisning, inklusive eventuella decimaler. Detta säkerställer att Procent-data exporteras korrekt medan användarens konfigurerade precision och formatering behålls.
