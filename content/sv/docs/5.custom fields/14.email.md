Det anpassade E-postfältet låter dig lagra e-postadresser inom dina Blue-poster. Att lägga till en e-post kommer att validera formatet för att säkerställa att en korrekt strukturerad e-post anges.

![Email 1](/docs/Email_1.png "Email 1")

Det anpassade e-postfältet i Blue
Varje post kan innehålla en e-postadress lagrad i ett enda E-postfält. Det finns funktionalitet för att klicka på den lagrade e-posten för att automatiskt öppna din standard e-postapplikation, vilket låter dig komponera e-post till associerade kontakter snabbt.

![Email 2](/docs/Email_2.png "Email 2")

Klicka för att öppna din standard e-postmjukvara

E-postfältet integreras sömlöst med [Blues E-postautomatiseringsmöjligheter](/sv/docs/automations/actions/email-automations). E-postarbetsflöden kan utlösas, och information från poster kan slås samman i e-post med hjälp av {merge} taggsyntax.

När du skapar ett E-postfält kan du namnge det kontextuellt för dina specifika behov, som "Support E-post" eller "Försäljnings E-post." Dock kan endast ett fält användas per post.

Med automatiserad validering, ett-klicks e-poståtkomst och arbetsflödesintegration låter E-postfältet dig lagra och ansluta med viktiga kontakter direkt från relevanta poster i Blue.
