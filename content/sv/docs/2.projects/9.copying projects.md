---
title: Kopiera projekt
description: Du kan skapa perfekta kopior av projekt i Blue.
---

Du kan kopiera projekt inom din organisation eller till andra organisationer genom att klicka på projektinställningsikonen. Endast [Projektadministratörer](/docs/user-management/roles/project-administrator) kan kopiera projekt.

![Copying Project 1](/docs/CopyingProject_1.png "Copying Project 1")

Observera att endast projektadministratörer har tillgång till projektinställningar

När du kopierar projekt kan du välja vad du vill kopiera.

![Copying Project 1](/docs/CopyingProject_2.png "Copying Project 2")

Välj vilka komponenter av ett projekt du vill kopiera

::callout
---

icon: lightbulb
---

**Tid som krävs för att kopiera projekt** <br/>
Beroende på storleken på ditt projekt kan kopiering ta allt från några sekunder till några minuter.
::
