---
title: Projekte kopieren
description: Sie können perfekte Kopien von Projekten in Blue erstellen.
---

Sie können Projekte innerhalb Ihrer Organisation oder in andere Organisationen kopieren, indem Sie auf das Symbol für Projekteinstellungen klicken. Nur [Projektadministratoren](/docs/user-management/roles/project-administrator) können Projekte kopieren.

![Copying Project 1](/docs/CopyingProject_1.png "Copying Project 1")

Beachten Sie, dass nur Projektadministratoren Zugriff auf die Projekteinstellungen haben

Beim Kopieren von Projekten können Sie auswählen, was Sie kopieren möchten.

![Copying Project 1](/docs/CopyingProject_2.png "Copying Project 2")

Wählen Sie aus, welche Komponenten eines Projekts Sie kopieren möchten

::callout
---

icon: lightbulb
---

**Erforderliche Zeit zum Kopieren von Projekten** <br/>
Je nach Größe Ihres Projekts kann das Kopieren von wenigen Sekunden bis zu einigen Minuten dauern.
::
