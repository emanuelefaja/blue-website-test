Les tags sont un moyen utile d'organiser votre travail. Vous pouvez les utiliser pour montrer différents types de tâches, l'urgence, ou toute catégorie qui convient à votre projet. Chaque tag peut être codé par couleur pour une identification facile et pour ajouter de la couleur à votre espace de travail.

<youtube url="https://www.youtube.com/watch?v=nlyY3cUUpkQ" />


## Utilisation des Tags

- **Ajouter des Tags** : Pour créer un tag, considérez les aspects de vos tâches qui nécessitent une catégorisation—que ce soit la nature de la tâche, comme 'bug', 'back-end', ou 'UI', ou sa priorité. Une fois que vous avez identifié vos catégories, vous pouvez facilement créer des tags et leur attribuer des couleurs uniques pour une reconnaissance immédiate.
- **Filtrer avec les Tags** : Les capacités de filtrage de Blue rendent la gestion de votre charge de travail plus intuitive. Vous pouvez filtrer les tags en utilisant l'icône de filtre déroulant ou directement dans la vue tableau en cliquant sur le tag désiré.

Tagging Cross-Projet <br/>
Pour les vues à l'échelle de l'organisation telles que les tableaux de bord et les cartes, le tagging devient particulièrement important car Blue fusionne automatiquement les tags du même nom. Vous n'avez donc qu'à filtrer par le tag "Priorité 1", qui affichera uniquement les enregistrements de tous les projets avec ce tag.
