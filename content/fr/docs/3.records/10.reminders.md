---
title: Rappels
description: Les rappels facilitent le suivi de vos tâches.
---

Les rappels dans Blue peuvent vous envoyer un e-mail ou une notification push mobile pour tout enregistrement à un moment prédéterminé. Qu'il s'agisse de répondre à un e-mail urgent, de soumettre un rapport ou de rappeler un client, vous pouvez programmer des rappels pour que vos responsabilités les plus pressantes soient toujours accomplies à temps. Nous nous chargeons d'envoyer les rappels - vous vous concentrez simplement sur l'accomplissement du travail.

## Définir un Rappel

Pour définir un rappel sur un enregistrement :
1. Ouvrez l'enregistrement
2. Cliquez sur l'icône d'horloge en haut à droite
3. Sélectionnez la date et l'heure souhaitées pour le rappel

![Reminder 1](/docs/Reminder_1.png "Reminder 1")

Ou par défaut, vous pouvez sélectionner parmi :

- Plus tard aujourd'hui
- Demain
- Plus tard cette semaine
- Ce week-end
- La semaine prochaine

![Reminder 2](/docs/Reminder_2.png "Reminder 2")

Une fois que vous avez défini un rappel, vous verrez la date et l'heure affichées directement sur l'enregistrement.

## Paramètres de Notification

Lorsque la date et l'heure du rappel arrivent, vous recevrez une notification par :
- E-mail (si activé dans vos paramètres de notification)
- Notification push dans les applications web/mobiles Blue (si activée)

La notification vous alertera pour revisiter l'enregistrement.

Dans vos paramètres de notification personnels, vous pouvez activer/désactiver les notifications par e-mail et push spécifiquement pour les rappels. Cela vous permet de choisir comment vous souhaitez être notifié pour les rappels.

Par défaut, les deux options sont activées lorsque les rappels sont configurés pour la première fois.

![Reminder 3](/docs/Reminder_3.png "Reminder 3")

## Modifier ou Supprimer des Rappels

Pour modifier ou supprimer un rappel :
1. Survolez la date sur l'enregistrement
2. Choisissez soit de modifier soit de supprimer le rappel

La modification vous permet de reporter le rappel à une date ultérieure. La suppression supprimera complètement le rappel.
