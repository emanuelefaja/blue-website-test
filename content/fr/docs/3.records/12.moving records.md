---
title: Déplacer les Enregistrements
description: Vous pouvez déplacer les enregistrements et toutes les informations associées entre les projets.
---

Parfois, vous pourriez avoir besoin de réorganiser vos enregistrements en les déplaçant entre les projets ou les listes. Voici comment procéder :

1. Naviguez vers l'enregistrement que vous souhaitez déplacer
2. Cliquez sur le menu "..."
3. Choisissez "Déplacer"
4. Sélectionnez le projet et la liste de destination

![Moving Record](/docs/MovingRecord.png "Moving Record")

Points à retenir :
- Seuls les administrateurs et les membres de l'équipe peuvent déplacer les enregistrements
- Lors du déplacement entre projets, les pièces jointes et commentaires seront transférés
- Cependant, les détails des champs personnalisés ne seront pas déplacés
