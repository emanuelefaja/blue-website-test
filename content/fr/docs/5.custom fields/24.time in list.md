Le champ personnalisé Temps en Liste suit automatiquement combien de temps un enregistrement a été dans chaque liste, fournissant des informations détaillées sur la progression du flux de travail et les goulots d'étranglement.

Le champ Temps en Liste est extrêmement précieux pour analyser l'efficacité du flux de travail et identifier les améliorations de processus. Quelques exemples incluent :

- Surveiller combien de temps les tickets de support passent dans chaque statut (Nouveau, En Cours, En Attente Client, Résolu)
- Suivre la progression du pipeline de vente pour identifier où les affaires se bloquent
- Analyser les étapes du flux de travail de fabrication pour optimiser les calendriers de production
- Mesurer les processus d'approbation de contenu du brouillon à la publication
- Identifier les goulots d'étranglement dans les flux de travail de recrutement de la candidature à l'embauche
- Suivre les phases de projet pour comprendre où les retards se produisent couramment
- Surveiller les étapes de livraison de service pour améliorer l'expérience client

Le champ Temps en Liste fournit un suivi automatique en temps réel sans intervention manuelle. Il calcule le temps passé dans chaque liste à la fois historiquement et actuellement, montrant des répartitions en pourcentage où les enregistrements passent le plus de temps. Cela permet une optimisation de processus basée sur les données, aide à identifier les inefficacités et fournit une visibilité claire sur les performances du flux de travail.

## Comment Ça Fonctionne

Le suivi du Temps en Liste commence automatiquement lorsque vous ajoutez le champ personnalisé Temps en Liste à votre projet. Le système :

1. **Suit les Heures d'Entrée et de Sortie** : Enregistre des horodatages précis lorsque les éléments entrent et sortent de chaque liste
2. **Calcule la Durée** : Calcule automatiquement le temps passé en utilisant la différence entre les heures d'entrée et de sortie
3. **Gère le Statut Actuel** : Pour les éléments actuellement dans une liste, calcule le temps depuis l'entrée jusqu'à maintenant
4. **Préserve l'Historique** : Maintient un suivi complet du temps même si les listes sont renommées ou supprimées
5. **Affiche la Répartition en Pourcentage** : Affiche quelle portion du temps total a été passée dans chaque liste

## Options d'Affichage

Les données de Temps en Liste apparaissent à plusieurs endroits :

- **Cartes d'Enregistrement** : Affiche le temps de liste actuel et le temps total du projet avec des icônes intuitives
- **Vue Détaillée d'Enregistrement** : Affiche une répartition complète avec des barres de progression pour chaque liste
- **Vue Base de Données** : Les données de temps peuvent être affichées en format tableau pour l'analyse
- **Intégration de Champ Personnalisé** : Fonctionne avec les champs personnalisés TIME_DURATION pour un suivi avancé

## Configuration

Le champ Temps en Liste ne nécessite aucune configuration - il suit automatiquement tous les mouvements de liste. Cependant, vous pouvez contrôler les options d'affichage :

- **Afficher le Temps dans la Liste Actuelle** : Basculer l'affichage du temps passé dans la liste actuelle sur les cartes d'enregistrement
- **Afficher le Temps dans le Projet** : Basculer l'affichage du temps total à travers toutes les listes du projet
- **Visibilité du Champ Personnalisé** : Contrôler si le suivi du temps apparaît dans les détails d'enregistrement

## Avantages Clés

- **Aucune Configuration Requise** : Commence automatiquement le suivi lorsque le champ est ajouté
- **Historique Complet** : Maintient le suivi même à travers les changements et suppressions de listes
- **Mises à Jour en Temps Réel** : Les calculs de temps se mettent à jour continuellement pour les éléments actuels
- **Indicateurs de Progression Visuels** : Les barres de progression montrent la distribution relative du temps à travers les listes
- **Optimisation des Processus** : Identifiez où les flux de travail ralentissent ou se bloquent
- **Métriques de Performance** : Générez des informations pour la productivité d'équipe et l'efficacité des processus
