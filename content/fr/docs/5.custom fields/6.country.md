Le champ personnalisé Pays vous permet d'associer des enregistrements dans Blue avec des pays spécifiques. Il fournit une liste déroulante contenant tous les pays reconnus dans le monde, vous permettant de sélectionner le pays pertinent pour chaque enregistrement.

![Country 1](/docs/Country_1.png "Country 1")

Le champ personnalisé Pays affiche un menu déroulant de tous les pays reconnus

Le menu déroulant affiche le nom du pays et l'icône du drapeau national, fournissant une identification visuelle claire. Vous pouvez sélectionner plusieurs pays par enregistrement si nécessaire.

![Country 2](/docs/Country_2.png "Country 2")

Les utilisateurs peuvent sélectionner plusieurs pays en même temps

Pour localiser rapidement un pays, commencez à taper le nom - le menu déroulant filtrera instantanément les pays correspondants dans une interface de recherche en temps réel. Cela rend la recherche même des noms de pays moins connus rapide et facile.

![Country 3](/docs/Country_3.png "Country 3")

Recherche instantanée pour n'importe quel pays

La liste complète est organisée par ordre alphabétique pour faciliter la consultation. Une fois sélectionné, le pays ou les pays choisis seront affichés en utilisant les icônes des drapeaux nationaux pour une référence visuelle continue.

Les utilisations potentielles du champ Pays incluent :

- Capturer les emplacements des clients, partenaires ou fournisseurs
- Associer des enregistrements avec des régions géographiques associées
- Filtrer les données par pays pour l'analyse et les rapports
- Suivre les destinations d'expédition internationale

Avec un support robuste pour tous les pays du monde et une interface de recherche raffinée, le champ déroulant Pays enrichit les enregistrements avec des données géographiques pertinentes.
