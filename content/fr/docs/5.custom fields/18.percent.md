Le champ personnalisé Pourcentage permet de capturer et calculer les valeurs de pourcentage dans les enregistrements Blue. Cela permet le suivi des progrès, la mesure des performances et la visualisation des données basée sur les pourcentages.

![Percent](/docs/Percent.png "Percent")

Le champ personnalisé Pourcentage est fourni avec une validation robuste pour soutenir des données précises. Lors de la saisie de valeurs, il vérifie que le nombre se trouve dans la plage standard de 0-100% attendue pour les pourcentages. Cela empêche l'introduction de données non valides.

Une fonctionnalité clé est la configuration flexible de la précision décimale. Vous pouvez définir le champ pour capturer les pourcentages avec autant de décimales que nécessaire selon vos besoins. Ce niveau de personnalisation permet le suivi de données basées sur des pourcentages à tout niveau de granularité requis par votre cas d'usage.

Pour simplifier les calculs, le champ Pourcentage s'intègre directement avec les [champs personnalisés de Formule](/fr/docs/custom-fields/formula). Cela facilite l'automatisation des barres de progression, des ratios, des taux et d'autres métriques communes basées sur les pourcentages. Les calculs se déroulent en arrière-plan, affichant la valeur de pourcentage de sortie.

Lors de l'exportation de données via les intégrations [CSV](/fr/docs/integrations/csv-export) ou [API](/fr/docs/integrations/api), le champ maintient son affichage de pourcentage formaté, y compris toutes les décimales. Cela garantit que les données de Pourcentage sont exportées avec précision tout en conservant la précision et le formatage configurés par l'utilisateur.
