Les champs personnalisés de type bouton vous permettent de créer des boutons qui peuvent être utilisés pour déclencher des actions d'automatisation dans Blue. C'est idéal pour créer des actions personnalisées, comme envoyer un e-mail, copier un enregistrement, ou déclencher une automatisation.

![Champ personnalisé Bouton](/product/buttom-custom-field.png "Champ personnalisé Bouton")

## Créer un champ personnalisé Bouton

Vous pouvez créer un champ personnalisé de type bouton de la même manière que vous créeriez tout autre champ personnalisé dans vos projets.

![Créer un champ personnalisé Bouton](/product/button-field-create.png "Créer un champ personnalisé Bouton")

Vous pouvez donner un nom à votre bouton et choisir également le type de confirmation.

## Confirmations

Les confirmations sont conçues pour éviter les clics accidentels sur les boutons en présentant aux utilisateurs une fenêtre contextuelle pour confirmer qu'ils souhaitent bien appuyer sur le bouton.

![Confirmation de bouton](/product/button-confirmation.png "Confirmation de bouton")

Vous avez trois options pour les confirmations :

1. **Aucune confirmation** — L'action d'automatisation est déclenchée immédiatement lors de l'appui sur le bouton.
2. **Confirmation simple** — Une fenêtre de confirmation s'affiche, et l'utilisateur doit simplement appuyer sur le bouton de confirmation
3. **Confirmation renforcée** — Une fenêtre de confirmation s'affiche, et l'utilisateur doit saisir une phrase spécifique puis appuyer sur le bouton de confirmation

Nous suggérons d'utiliser les confirmations renforcées pour les boutons liés à des actions d'automatisation qui déclenchent des actions importantes, comme l'envoi d'un e-mail de confirmation à un client.

## Lier les boutons aux actions d'automatisation

Vous pouvez sélectionner un bouton spécifique comme action pouvant déclencher toute automatisation prise en charge dans Blue.

![Action de bouton](/product/button-action.png "Action de bouton")
