Le champ personnalisé Case à cocher fournit une case à cocher qui peut être utilisée à diverses fins, notamment pour les approbations ou [pour déclencher une automation](/fr/docs/automations/introduction)

![Checkbox 2](/docs/Checkbox_1.png "Checkbox 1")

Utilisation des champs personnalisés Case à cocher pour les approbations

Les approbations constituent un cas d'usage particulièrement adapté pour le champ personnalisé Case à cocher grâce aux capacités d'audit de Blue. Chaque fois que la case est cochée ou décochée, Blue enregistre l'action de manière permanente :

![Checkbox 2](/docs/Checkbox_2.png "Checkbox 2")

Blue enregistre la personne, l'action et l'heure pour chaque modification.
