Le champ personnalisé Nombre permet de capturer des valeurs de données numériques dans des limites minimales et maximales configurées.

![Number](/docs/Number.png "Number")

Les [Administrateurs de Projet](/fr/docs/user-management/roles/project-administrator) peuvent définir la plage autorisée lors de la configuration du champ personnalisé Nombre. Le système validera les entrées pour s'assurer qu'elles sont dans les valeurs minimales et maximales autorisées.
