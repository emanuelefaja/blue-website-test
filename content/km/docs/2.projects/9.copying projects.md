អ្នកអាចចម្លងគម្រោងនៅក្នុងអង្គការរបស់អ្នក ឬទៅអង្គការផ្សេងទៀតដោយចុចលើរូបតំណាងការកំណត់គម្រោង។ មានតែ[អ្នកគ្រប់គ្រងគម្រោង](/docs/user-management/roles/project-administrator)ប៉ុណ្ណោះដែលអាចចម្លងគម្រោងបាន។

![Copying Project 1](/docs/CopyingProject_1.png "Copying Project 1")

សូមចំណាំថា មានតែអ្នកគ្រប់គ្រងគម្រោងប៉ុណ្ណោះដែលមានសិទ្ធិចូលប្រើការកំណត់គម្រោង

ពេលចម្លងគម្រោង អ្នកអាចជ្រើសរើសអ្វីដែលអ្នកចង់ចម្លង។

![Copying Project 1](/docs/CopyingProject_2.png "Copying Project 2")

ជ្រើសរើសធាតុផ្សំណាមួយនៃគម្រោងដែលអ្នកចង់ចម្លង

::callout
---

icon: lightbulb
---

**ពេលវេលាដែលត្រូវការសម្រាប់ចម្លងគម្រោង** <br/>
អាស្រ័យលើទំហំនៃគម្រោងរបស់អ្នក ការចម្លងអាចចំណាយពេលពីពីរបីវិនាទីទៅពីរបីនាទី។
::

អ្នកអាចមើលវីដេអូអំពីការចម្លងគម្រោង៖
