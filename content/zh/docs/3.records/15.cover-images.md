## 概述

封面图片允许您通过直接在记录卡片上显示图片预览来为您的记录添加可视化上下文。此功能会自动从记录内的各种来源提取图片并将其显示为缩略图，使您能够一目了然地识别和组织视觉内容。

无论您是在管理设计项目、内容日历、产品目录还是任何涉及视觉资产的工作流程，封面图片都能帮助您快速浏览记录并找到所需内容。该功能包括实时进度跟踪和灵活的显示选项，以匹配您的特定工作流程需求。

## 主要功能

- **自动图片提取**: Blue 自动从您选择的来源查找并显示图片
- **多种图片来源**: 从记录描述、评论或自定义文件字段中提取图片
- ** 显示适配选项**: 选择图片如何显示，包括覆盖或包含模式
- **智能选择**: 配置是使用来源中的第一张还是最后一张图片
- **实时更新**: 当记录内容发生变化时，封面图片会自动更新
- **实时进度跟踪**: 通过实时进度订阅监控批量处理

## 工作原理

### 启用封面图片

封面图片由项目管理员在项目级别进行配置：

1. 打开您的项目并导航到**项目设置**
2. 选择**记录设置**
3. 找到**图片封面选项**部分
4. 切换**启用封面图片**以激活该功能

### 配置选项

#### 图片来源
选择 Blue 应该在何处查找图片：
- **描述**: 从记录描述中嵌入的图片中提取（默认）
- **评论**: 从记录评论中拉取第一张/最后一张图片
- **自定义字段**: 使用特定文件类型自定义字段中的文件（支持任何文件类型）

#### 图片选择
当有多张图片可用时，选择要显示的图片：
- **第一张图片**: 使用找到的第一张图片（默认）
- **最后一张图片**: 使用最新的图片

#### 显示适配
控制图片在预览区域中的显示方式：
- **覆盖**: 图片填充整个预览区域，可能会裁剪边缘以保持纵横比（默认）
- **包含**: 在预览区域内显示完整图片，可能有空白区域

## 使用案例

**设计项目** - 直接在任务卡片上显示模型和设计迭代，轻松查看每个设计任务的当前状态，无需打开单个记录。

**内容规划** - 对于内容日历和编辑工作流程，显示特色图片或缩略图以可视化您的内容管道。

**产品管理** - 在库存记录或功能请求上显示产品照片，帮助团队在讨论期间快速识别项目。

**房地产** - 在房源记录上显示物业照片，允许经纪人快速浏览视觉化物业。

**活动策划** - 在规划任务上显示场地照片、设计概念或活动布局，以实现更好的视觉组织。

## 设置封面图片

### 步骤 1: 选择您的图片来源

考虑图片在您的工作流程中自然存在的位置：
- 如果图片是任务描述或简报的一部分，使用**描述**
- 如果图片通过团队协作添加，使用**评论**
- 如果您有专用的图片字段，使用**自定义字段**

### 步骤 2: 配置显示设置

1. 根据您的图片类型选择首选的**图片适配**：
   - 对于可接受裁剪的照片和设计使用**覆盖**
   - 对于需要查看全部内容的图表或屏幕截图使用**包含**

2. 根据您的工作流程选择**图片选择**：
   - 对于主要图片最初添加的工作流程使用**第一张**
   - 对于最新图片最相关的工作流程使用**最后一张**

### 步骤 3: 处理现有记录

当您首次启用封面图片或更改配置时，Blue 会自动处理您的记录：
1. 实时进度更新显示处理状态
2. 记录以批次处理（每批次 50 条记录）以获得最佳性能
3. 您可以在后台处理进行时继续工作
4. 处理完成时列表会自动刷新
5. 保存配置更改时会自动触发处理

## 技术详情

### 支持的格式
- 使用描述或评论来源时：
  - JPEG/JPG 图片
  - PNG 图片（包括透明度）
  - GIF 图片（静态显示）
- 使用自定义字段来源时：
  - 可以显示任何文件类型（图片、PDF、文档）
  - 非图片文件显示相应的文件类型图标
- 所有文件必须上传到 Blue（不支持外部 URL）

### 限制
- 每个项目最多 6,000 条记录用于批量封面生成
- 只有文件类型自定义字段可以用作来源
- 封面图片为只读，无法直接编辑

### 自动更新
封面在以下情况下自动重新生成：
- 记录描述被编辑时（如果使用描述来源）
- 添加带有图片的新评论时（如果使用评论来源）
- 自定义字段中的文件被更新时（如果使用自定义字段来源）
- 封面配置设置被更改时
- 修改显示适配选项时

### API 访问
封面图片完全集成到 GraphQL API 中：
- 通过 `setTodoCoverConfig` 突变进行配置
- 通过 Todo 类型上的 `cover` 字段访问封面数据
- 使用 `subscribeToCoverGenerationProgress` 订阅监控进度
- 缩略图 URL 自动在 `/uploads/{uid}/thumbnail/cover` 生成
