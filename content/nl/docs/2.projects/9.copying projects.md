---
title: Projecten kopiëren
description: Je kunt perfecte kopieën van projecten maken in Blue.
---

Je kunt projecten kopiëren binnen je organisatie of naar andere organisaties door op het projectinstellingen-icoon te klikken. Alleen [Projectbeheerders](/docs/user-management/roles/project-administrator) kunnen projecten kopiëren.

![Copying Project 1](/docs/CopyingProject_1.png "Copying Project 1")

Let op dat alleen projectbeheerders toegang hebben tot projectinstellingen

Bij het kopiëren van projecten kun je kiezen wat je wilt kopiëren.

![Copying Project 1](/docs/CopyingProject_2.png "Copying Project 2")

Kies welke onderdelen van een project je wilt kopiëren

::callout
---

icon: lightbulb
---

**Benodigde tijd om projecten te kopiëren** <br/>
Afhankelijk van de grootte van je project kan kopiëren enkele seconden tot enkele minuten duren.
::
