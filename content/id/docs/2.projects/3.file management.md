Anda kemungkinan akan memiliki file dan dokumen pendukung selama proyek atau proses berlangsung. Ini bisa berupa file desain yang sedang dalam pengerjaan, dokumen word untuk ditinjau, gambar, dan video.

Blue menawarkan penyimpanan file yang besar, [sesuai dengan kebijakan penggunaan wajar kami](https://www.blue.cc/legal/terms), untuk semua kebutuhan proyek dan proses Anda. Ukuran file maksimum untuk file individual adalah 5GB.

## Mengunggah dalam Komentar

Anda dapat mengunggah file di mana pun Anda dapat menulis komentar; file ditampilkan sejajar dengan teks Anda alih-alih sebagai lampiran pada pesan Anda. Kami percaya ini penting agar setiap orang memiliki konteks yang tepat dari pesan Anda dengan data di ujung jari mereka.

![File Management 1](/docs/file_1.png "File Management 1")

Blue menempatkan file Anda langsung ke dalam konteks — tidak tersembunyi.

File juga akan ditampilkan di tab file. Perhatikan bahwa jika komentar asli tempat file diunggah dihapus, file juga dihapus dari tab file.

Jika Anda ingin mengunduh file ke komputer Anda, Anda dapat mengklik kanan pada file dan kemudian memilih opsi unduh.

## Mencari File

Anda dapat mencari file berdasarkan namanya menggunakan bilah pencarian file. Ini akan langsung mencari di semua file di semua folder dan sub-folder.

## Menemukan Lokasi File

Di tab File, Anda dapat dengan mudah menemukan catatan yang berisi file tertentu. Cukup klik kanan pada file dan pilih "Temukan File" dari menu konteks. Tindakan ini akan membawa Anda langsung ke catatan yang menyimpan file tersebut, membuatnya mudah untuk mengakses data terkait.

![File Management 8](/docs/file_8.png "File Management 8")

## Memindahkan File

Untuk mengatur file Anda ke dalam folder masing-masing, ikuti langkah-langkah berikut:
1. Klik kanan pada file yang ingin Anda pindahkan.
2. Dari menu konteks, pilih "Pindahkan ke folder" dan menampilkan daftar folder yang tersedia.
3. Pilih folder yang diinginkan tempat Anda ingin memindahkan file.
4. Jika Anda perlu memindahkan beberapa file secara bersamaan, tahan tombol "Shift" saat memilih file sebelum mengklik kanan.

Proses ini memungkinkan Anda untuk dengan mudah memindahkan file ke folder yang ditentukan, menjaga data Anda tetap teratur dan dapat diakses.

![File Management 9](/docs/file_9.png "File Management 9")

## Berbagi File

Di Blue, Anda dapat berbagi file secara publik dan menghasilkan tautan yang dapat diunduh untuk diakses oleh siapa saja. Berikut caranya:

1. Navigasi ke file yang ingin Anda bagikan.
2. Pilih file dengan mengkliknya.
3. Klik tombol "Bagikan".
4. Tautan unik yang dapat dibagikan akan dihasilkan untuk file yang dipilih.
5. Salin dan bagikan tautan ini dengan penerima yang dituju.
6. Setelah mereka mengunduh file, Anda dapat menonaktifkan opsi berbagi untuk menjaga keamanan file.

Fitur ini memungkinkan Anda berbagi file secara publik dengan mudah, sambil juga memberi Anda kontrol untuk mencabut akses bila diperlukan. Ingatlah untuk menonaktifkan berbagi setelah file diperoleh untuk melindungi data Anda.

## Folder

### Membuat Folder

Folder adalah cara yang berharga untuk menjaga file Anda tetap teratur. [Administrator Proyek](/docs/user-management/roles/project-administrator) dan [Anggota Tim](/docs/user-management/roles/team-member) dapat membuat folder. Anda dapat membuat folder dan sub-folder dalam jumlah tak terbatas untuk mencocokkan struktur file organisasi Anda.

Anda dapat membuat folder dengan mengklik kanan pada ruang kosong mana pun di tab file.

![File Management 11](/docs/file_11.png "File Management 11")

Klik kanan pada ruang kosong mana pun di tab file dan pilih "Folder Baru"

Sebagai alternatif, Anda juga dapat mengklik tombol navigasi "File" di kiri atas:

Mengganti nama folder itu mudah: cukup klik kanan pada folder dan pilih ganti nama. Hanya [Administrator Proyek](/docs/user-management/roles/project-administrator) dan [Anggota Tim](/docs/user-management/roles/team-member) yang dapat mengganti nama folder.

## Tindakan Massal

Untuk menghemat waktu Anda, Blue mendukung banyak tindakan massal untuk manajemen file.

- **Unggah —** Seret dan lepas beberapa file secara bersamaan atau bahkan seluruh folder untuk diunggah. Jika Anda mengunggah file besar, Anda dapat terus menggunakan Blue seperti biasa dan file akan diunggah di latar belakang.
- **Shift-Select —** Anda dapat memilih beberapa file secara bersamaan dengan menahan SHIFT dan memilih file yang Anda inginkan.
- **Unduh —** Menggunakan Shift-Select, Anda kemudian dapat mengklik kanan untuk mengunduh semua file secara bersamaan. Blue akan mengunduh setiap file secara individual di browser Anda.
- **Pindahkan —** Anda dapat memindahkan file ke folder atau subfolder dengan menggunakan Shift-Select, Anda kemudian dapat mengklik kanan, pilih pindahkan dan kemudian pilih folder pilihan Anda. Blue tidak mendukung pemindahan file antar proyek.
- **Hapus —** Menggunakan Shift-Select, Anda kemudian dapat mengklik kanan untuk menghapus semua file secara bersamaan. Blue akan menampilkan popup peringatan/konfirmasi sebelum menghapus file apa pun. Blue akan menghapus semua file dalam folder jika Anda menghapus folder.
