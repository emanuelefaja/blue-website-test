Круговые диаграммы в Blue предоставляют интуитивно понятный круговой график для визуализации пропорциональных данных по категоризированным записям. Эти графики иллюстрируют относительные величины и взаимосвязи в наборах данных, используя цветные, сегментированные срезы.

## Создание круговой диаграммы

### Установите заголовок

При создании круговой диаграммы первым делом необходимо установить заголовок для вашей диаграммы, а затем получить данные для значений ваших срезов. Отдельные срезы могут содержать вложенные подгруппы, что позволяет проводить многоуровневый анализ в одной диаграмме. Например, объедините региональные и каналовые деления внутри каждого среза продукта.

![Круговая диаграмма 1](/docs/dashboard/pieChart/PieChart_1.png "Pie Chart 1")

### Создание значений или нескольких значений.

Вы можете создать несколько срезов для диаграммы, которые будут формировать ваш круг. Каждый срез должен извлекать данные из пользовательских полей других проектов, чтобы иметь смысл. Если вы выбираете данные из [пользовательского поля "Валюта"](/docs/custom-fields/currency), остальные срезы для этого должны извлекаться из тех же данных. Это позволяет вам сравнивать одни и те же данные между проектами в одной круговой диаграмме.

Значения на круговой диаграмме могут быть получены из конкретного проекта, позволяя вам делать следующее:

- Анализировать состав продаж по регионам, каналам, рынкам и т.д.
- Разбивать распределение задач или проблем по категориям, приоритетам, исполнителям и т.д.
- Понимать прогресс стадий рабочего процесса и узкие места
- Сравнивать коэффициенты дохода или прибыли по продуктам, кампаниям, секторам и т.д.
- Отслеживать метрики разнообразия и инклюзии по командам и уровням

![Круговая диаграмма 2](/docs/dashboard/pieChart/PieChart_2.png "Pie Chart 2")

Чтобы гарантировать, что значения на вашей круговой диаграмме являются максимально актуальными и целевыми, Blue предлагает мощные варианты фильтрации — [любой фильтр](/docs/views/filters) в любом представлении проекта доступен для использования!

### Добавить логику

В Blue функциональность логики в статистических карточках является движком для вычисления и представления основанных на данных инсайтов. Логика позволяет вам использовать силу данных через настраиваемые формулы и операции, преобразуя сырые данные проекта в полезную информацию.

Начните с обращения к конкретным значениям в вашем проекте, набрав '{'. Это действие вызовет список доступных значений, которые вы можете включить в свою логическую формулу.

![Круговая диаграмма 3](/docs/dashboard/pieChart/PieChart_3.png "Pie Chart 3")

Вы можете улучшить свою логику с помощью различных функций, набрав '='. Blue предоставляет набор функций, таких как:

- **AVERAGE**: Вычислить среднее значение набора чисел.
- **AVERAGEA**: Определить среднее, рассматривая текст как ноль.
- **COUNT**: Подсчитать количество записей в числовом поле.
- **COUNTA**: Подсчитать количество непустых значений.
- **MAX**: Найти максимальное значение в наборе.
- **MIN**: Определить минимальное значение в наборе.
- **MINUS**: Вычесть одно значение из другого.
- **MULTIPLY**: Умножить два значения.
- **SUM**: Сложить серию чисел.

![Круговая диаграмма 4](/docs/dashboard/pieChart/PieChart_4.png "Pie Chart 4")

::callout
 Вы можете выбрать цвет для каждого среза круговой диаграммы. ::

Эти функции являются важными инструментами в анализе данных, позволяя вам эффективно обобщать и анализировать метрики проекта. Независимо от того, хотите ли вы определить максимальные продажи, рассчитать среднюю продолжительность задач или суммировать бюджетные allocations, логика в Blue делает это простым.

![Круговая диаграмма 5](/docs/dashboard/pieChart/PieChart_5.png "Pie Chart 5")

Круговые диаграммы преобразуют данные проекта в полезные и визуально интуитивные инсайты о пропорциональных вкладах, прогрессе и различиях по категоризированным записям в Blue.

::callout
