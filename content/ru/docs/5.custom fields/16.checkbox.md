Пользовательское поле Checkbox предоставляет флажок, который может использоваться для различных целей, включая утверждения или [для запуска автоматизации](/ru/docs/automations/introduction)

![Checkbox 2](/docs/Checkbox_1.png "Checkbox 1")

Использование пользовательских полей Checkbox для утверждений

Утверждения являются особенно хорошим случаем использования для пользовательского поля Checkbox благодаря возможностям аудита Blue. Каждый раз, когда флажок устанавливается или снимается, Blue регистрирует действие постоянным образом:

![Checkbox 2](/docs/Checkbox_2.png "Checkbox 2")

Blue регистрирует человека, действие и время для каждого изменения.
