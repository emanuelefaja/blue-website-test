Пользовательское поле "Проценты" позволяет захватывать и вычислять процентные значения в записях Blue. Это обеспечивает отслеживание прогресса, измерение производительности и визуализацию данных на основе процентов.

![Percent](/docs/Percent.png "Percent")

Пользовательское поле "Проценты" поставляется с надежной валидацией для поддержки точных данных. При вводе значений оно проверяет, что число находится в стандартном диапазоне 0-100%, ожидаемом для процентов. Это предотвращает введение недействительных данных.

Ключевой особенностью является гибкая настройка десятичной точности. Вы можете настроить поле для захвата процентов с таким количеством десятичных знаков, сколько необходимо на основе ваших потребностей. Этот уровень кастомизации позволяет отслеживать данные на основе процентов на любом уровне детализации, который требует ваш случай использования.

Для упрощения расчетов поле "Проценты" интегрируется напрямую с [пользовательскими полями Формула](/ru/docs/custom-fields/formula). Это делает легкой автоматизацию полос прогресса, соотношений, ставок и других общих метрик на основе процентов. Расчеты происходят за кулисами, отображая выходное процентное значение.

При экспорте данных через интеграции [CSV](/ru/docs/integrations/csv-export) или [API](/ru/docs/integrations/api), поле сохраняет свое форматированное процентное отображение, включая любые десятичные знаки. Это обеспечивает точный экспорт данных "Проценты" при сохранении настроенной пользователем точности и форматирования.
