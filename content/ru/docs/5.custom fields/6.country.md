Настраиваемое поле "Страна" позволяет связывать записи в Blue с конкретными странами. Оно предоставляет выпадающий список, содержащий все признанные страны мира, позволяя выбрать соответствующую страну для каждой записи.

![Country 1](/docs/Country_1.png "Country 1")

Настраиваемое поле "Страна" отображает выпадающий список всех признанных стран

Выпадающее меню отображает название страны и значок национального флага, обеспечивая четкую визуальную идентификацию. При необходимости можно выбрать несколько стран для одной записи.

![Country 2](/docs/Country_2.png "Country 2")

Пользователи могут одновременно выбрать несколько стран

Для быстрого поиска страны начните вводить название - выпадающий список мгновенно отфильтрует совпадающие страны в интерфейсе поиска по мере ввода. Это делает поиск даже малоизвестных названий стран быстрым и легким.

![Country 3](/docs/Country_3.png "Country 3")

Мгновенный поиск любой страны

Полный список организован в алфавитном порядке для удобства просмотра. После выбора выбранная страна или страны будут отображаться с использованием значков национальных флагов для постоянной визуальной справки.

Потенциальные варианты использования поля "Страна" включают:

- Фиксация местоположения клиентов, партнеров или поставщиков
- Связывание записей с соответствующими географическими регионами
- Фильтрация данных по странам для анализа и отчетности
- Отслеживание международных направлений доставки

Благодаря надежной поддержке всех стран мира и усовершенствованному интерфейсу с возможностью поиска, выпадающее поле "Страна" обогащает записи соответствующими географическими данными.
