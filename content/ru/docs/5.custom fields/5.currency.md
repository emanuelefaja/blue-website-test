Храните финансовую информацию в поле валюты

При добавлении валютных значений поле будет форматировать число согласно местным соглашениям и отображать соответствующий символ валюты. Например, $1,000.50 для долларов США или €1.500,25 для евро.

Поле валюты проверяет вводимые данные, чтобы обеспечить захват только действительных денежных сумм. После добавления в проект поле будет хранить суммы в одной выбранной валюте. Оно не предоставляет автоматическое конвертирование валют между различными номиналами.

Это поле обеспечивает оптимизированный опыт для отслеживания денежных данных, связанных с записями. Автоматическое форматирование устраняет сложность ручного ввода валютных значений в местных форматах, а проверка защищает целостность данных.

Потенциальные области применения поля валюты включают:

- Захват сумм транзакций
- Хранение бюджетов проектов
- Отслеживание общих сумм счетов-фактур
- Запись сумм платежей
- Расчёт доходов

Благодаря надёжной поддержке глобальных валютных форматов и упрощённому вводу данных пользовательское поле "Валюта" позволяет централизовать ключевую финансовую информацию в ваших записях в Blue.
