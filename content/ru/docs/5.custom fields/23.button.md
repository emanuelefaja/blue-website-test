Пользовательские поля кнопок позволяют создавать кнопки, которые могут использоваться для запуска автоматических действий в Blue. Это отлично подходит для создания пользовательских действий, таких как отправка электронной почты, копирование записи или запуск автоматизации.

## Создание пользовательского поля кнопки

Вы можете создать пользовательское поле кнопки таким же образом, как и любое другое пользовательское поле в ваших проектах.

![Create Button Custom Field](/product/button-field-create.png "Create Button Custom Field")

Вы можете дать своей кнопке имя, а также выбрать тип подтверждения.

## Подтверждения

Подтверждения предназначены для предотвращения случайных нажатий кнопок путем представления пользователям всплывающего окна для подтверждения того, что они действительно хотят нажать кнопку.

![Button Confirmation](/product/button-confirmation.png "Button Confirmation")

У вас есть три варианта подтверждений:

1. **Без подтверждения** — Автоматическое действие запускается немедленно при нажатии кнопки.
2. **Мягкое подтверждение** — Показывается всплывающее окно подтверждения, и пользователю просто нужно нажать кнопку подтверждения
3. **Жесткое подтверждение** — Показывается всплывающее окно подтверждения, и пользователю нужно ввести определенную фразу, а затем нажать кнопку подтверждения

Мы рекомендуем использовать жесткие подтверждения для кнопок, которые связаны с автоматическими действиями, запускающими важные действия, такие как отправка клиенту подтверждающего электронного письма.

## Привязка кнопок к автоматическим действиям

Вы можете выбрать определенную кнопку в качестве действия, которое может запускать любую автоматизацию, поддерживаемую в Blue.

![Button Action](/product/button-action.png "Button Action")
