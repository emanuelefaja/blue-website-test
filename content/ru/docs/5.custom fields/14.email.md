Пользовательское поле Электронная почта позволяет хранить адреса электронной почты в ваших записях Blue. Добавление электронной почты проверит формат, чтобы убедиться, что введен правильно структурированный электронный адрес.

![Email 1](/docs/Email_1.png "Email 1")

Пользовательское поле электронной почты в Blue
Каждая запись может содержать один адрес электронной почты, хранящийся в одном поле Электронная почта. Есть функциональность для клика по сохраненной электронной почте, чтобы автоматически открыть ваше приложение электронной почты по умолчанию, позволяя вам быстро создавать электронные письма связанным контактам.

![Email 2](/docs/Email_2.png "Email 2")

Нажмите, чтобы открыть программу электронной почты по умолчанию

Поле Электронная почта легко интегрируется с [возможностями Автоматизации электронной почты Blue](/ru/docs/automations/actions/email-automations). Рабочие процессы электронной почты могут быть запущены, а информация из записей может быть объединена в электронные письма с использованием синтаксиса тега {merge}.

При создании поля Электронная почта вы можете назвать его контекстуально для ваших конкретных потребностей, таких как "Электронная почта поддержки" или "Электронная почта продаж." Однако только одно поле может использоваться на запись.

С автоматической проверкой, доступом к электронной почте одним кликом и интеграцией рабочего процесса, поле Электронная почта позволяет вам хранить и связываться с важными контактами непосредственно из соответствующих записей в Blue.
