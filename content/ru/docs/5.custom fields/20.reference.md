Пользовательские поля типа "Ссылка" позволяют связывать записи между различными проектами. Они предоставляют выпадающий список, который позволяет выбрать запись из другого проекта.
Пользовательские поля типа "Ссылка" работают совместно с [пользовательскими полями типа "Поиск"](/ru/docs/custom-fields/lookup), чтобы переносить информацию из одного проекта в другой. Это чрезвычайно мощный инструмент, так как он открывает множество сценариев использования в Blue.

Компания ACME использует пользовательские поля "Ссылка" и "Поиск" Blue для создания взаимосвязанной экосистемы данных в отдельных проектах "Клиенты", "Продажи" и "Склад". Записи клиентов в проекте "Клиенты" связаны через поле "Ссылка" с транзакциями продаж в проекте "Продажи", что позволяет полям "Поиск" подтягивать связанные данные клиентов, такие как номер телефона и статус аккаунта, прямо в каждую запись продажи; дополнительно, проданные товары могут отображаться в записи продажи через поле "Поиск", ссылающееся на данные "Количество продано" из проекта "Склад"; наконец, списания со склада связаны с соответствующими продажами через поле "Ссылка" в проекте "Склад", указывающее обратно на записи продаж, обеспечивая полную видимость того, какая продажа вызвала списание со склада. Используя поля "Ссылка" и "Поиск", ACME связывает клиентов, продажи и склад в интегрированное 360-градусное представление между проектами.

## Настройка

При создании поля "Ссылка" [Администратор проекта](/ru/docs/user-management/roles/project-administrator) выбирает конкретный проект, который будет предоставлять список ссылочных записей.

Затем они настраивают, будет ли поле разрешать:
- Одиночный выбор: Выбрать одну ссылочную запись
- Множественный выбор: Выбрать несколько ссылочных записей

Наконец, можно установить фильтр, чтобы позволить пользователям выбирать только записи, соответствующие фильтру.

## Отслеживание ссылающихся записей.

Иногда, когда у вас есть записи, на которые ссылаются в нескольких проектах, вы хотите мгновенно видеть, где на них ссылались, все в одном месте. Вы можете сделать это, настроив пользовательское поле "Ссылающиеся на".

Это позволяет видеть, где ваша запись была использована в качестве ссылки в разных проектах, чтобы у вас был след того, где на нее ссылались. Одним щелчком вы будете мгновенно перенесены в этот конкретный проект.

![Reference 3](/docs/Reference_3.png "Reference 3")

Поле "Ссылающиеся на" автоматически заполнится всеми записями, чтобы показать, где на них ссылались

![Reference 4](/docs/Reference_4.png "Reference 4")

Эта функция особенно полезна, когда вы хотите видеть, как ваш список контактов клиентов используется в нескольких проектах, и обеспечивает быстрый и простой способ отслеживать использование записей между проектами.

Чтобы удалить это поле "Ссылающиеся на" из вашей записи:

1. Перейдите в "Управление полями"
2. Нажмите красную корзину рядом с полем "Ссылающиеся на", чтобы удалить его.
