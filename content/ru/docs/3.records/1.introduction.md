## Обзор

Записи в Blue служат двум ключевым целям:

1. **Отслеживание рабочих элементов**: Вы можете использовать записи как динамические списки дел для отслеживания прогресса задач, проектов, продаж и т.д. Назначайте сроки выполнения, теги, чек-листы и пользовательские поля для сбора всей релевантной информации.
2. **Организация информации**: Записи позволяют Вам структурировать информацию, используя списки как категории. Например, у Вас может быть список "Открытые лиды" и "Закрытые сделки" для организации Вашей воронки продаж.

## Ключевые функции

Вот некоторые из ключевых функций записей в Blue:
- [Списки](/docs/records/lists): Сортируйте записи по различным спискам для категоризации информации. Например, создайте списки "К выполнению", "В процессе" и "Завершено".
- [Пользовательские поля](/docs/custom-fields/introduction): Добавляйте пользовательские поля, такие как выпадающие списки, числа и даты, для сбора специализированной информации. Например, добавьте числовое поле "Оценка лида".
- [Теги](/docs/records/tags): Используйте цветные теги, такие как "Срочно" или "Ошибка", для выделения приоритетных записей.
- [Комментарии](/docs/records/comments): Сотрудничайте с Вашей командой, оставляя комментарии непосредственно к записям.
- [Вложения](/docs/projects/file-management): Добавляйте файлы, изображения, видео непосредственно к записям для удобного доступа.
- [Автоматизации](/docs/automations/introduction): Запускайте автоматизации при создании, редактировании или удалении записей. Например, отправляйте email-уведомления при создании записи с высоким приоритетом.
- **Цвета**: Устанавливайте цвета для Ваших записей, чтобы определить срочность записей или для планирования контента.

## Случаи использования

Вот несколько примеров того, как Вы можете использовать записи в Вашей организации:

### Управление задачами и проектами

Используйте записи для отслеживания прогресса задач и проектов с исполнителями, сроками выполнения, вложениями и т.д. Группируйте их в списки, такие как "Бэклог", "В процессе" и "Запущено", для визуализации рабочего процесса.

### Воронка продаж

Управляйте Вашей воронкой продаж, создавая записи для каждого лида и возможности. Отслеживайте стоимость сделки, оценку лида и следующие шаги в пользовательских полях. Организуйте записи в списки, такие как "Открытые сделки" и "Закрытые выигранные", для анализа Вашей воронки.

### Тикеты поддержки

Записи могут служить тикетами поддержки для отслеживания проблем клиентов. Агентам могут быть назначены тикеты, они могут задавать уточняющие вопросы в комментариях и обновлять статус в пользовательском поле. Группируйте тикеты по типу, используя теги, такие как "Ошибка", "Вопрос" и т.д.

### Рекрутинг

В Вашем процессе найма создайте запись для каждого заявления кандидата с ключевой информацией в пользовательских полях. Перемещайте их через списки, такие как "Подано заявление", "Телефонное интервью" и "Финальный раунд", для отслеживания прогресса. Используйте email-автоматизацию для отправки обновлений кандидату при перемещении записи в другой список.

Таким образом, записи предоставляют Вам гибкий способ создания пользовательского рабочего процесса, который соответствует Вашему бизнес-процессу. С такими функциями, как списки, пользовательские поля и автоматизации, Вы можете создавать мощные системы на основе записей в Blue.
