[Make.com](https://www.make.com) 是一個整合平台，幫助連接不同的網路應用程式並自動化工作流程。Blue 與 Make.com **正式整合**，實現 2,000 多個應用程式之間的無縫自動化—無需編碼！

![Blue Make.com 模組概覽](/docs/blue_make_modules_overview.png "Overview of available Blue modules in Make.com")

## 使用案例

以下是一些強大的方法，可以通過 Make.com 整合擴展 Blue 的功能：
Blue 的 Make.com 整合使各種業務功能之間的強大自動化成為可能。對於客戶體驗，您可以將 Blue 與 Intercom、HubSpot 和 Zendesk 等工具連接，以創建全面的客戶記錄、同步互動並通過 Slack 和 Microsoft Teams 維持實時通信。在項目管理方面，Blue 與 Jira、GitHub、Trello 和 Monday.com 整合，以同步任務、跟踪進度並自動化通知。

在行銷和銷售方面，Blue 與 Mailchimp、Stripe 和 Calendly 連接，以跟踪活動、管理付款並自動化會議工作流程。通過與 Google Drive、Dropbox、OneDrive 和 SharePoint 的整合，文檔管理變得更加流暢，而 Slack、Gmail 和 WhatsApp 等通信工具則通過 Blue 的中央任務管理系統統一。數據分析能力通過與 Google Sheets、Power BI、Airtable 和 Tableau 的連接得以增強，並且針對金融服務、電子商務和遠程團隊管理提供行業特定的解決方案。

## 連接並驗證您的 Blue 帳戶

要安全地將 Blue 與 Make.com 連接：

1. 登錄到您的 Blue 帳戶並打開您的個人資料 → 轉到 **API** 標籤
2. 點擊 **生成令牌**，提供名稱（和可選的過期時間）
3. 複製以下兩項：
   - **令牌 ID** (`x‑bloo‑token‑id`)
   - **密鑰** (`x‑bloo‑token‑secret`)

::callout
 ---
icon: lightbulb
---
**重要提示** <br/>
密鑰僅顯示一次—請安全存儲！
::

5. 在 Make.com 中，創建新連接時，輸入：
   - **令牌 ID**
   - **密鑰**

## 模組參數參考

### 監視事件
- 事件（可選）：多選事件類型
- 公司 ID（可選）：多選公司 ID 或別名
- 項目 ID（可選）：多選項目 ID 或別名

### 創建記錄
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 列表：列表 ID
- 記錄名稱：記錄的名稱
- 默認字段：默認值
- 單一日期：截止日期
- 日期範圍：開始和截止日期
- 被指派者：多選用戶
- 標籤：多選標籤
- 描述：文本字段，支持 HTML
- 清單：清單標題的數組

**支持的自定義字段：**
| 字段類型 | 描述 |
|------------|-------------|
| 單行文本 | 單行文本字段，具有可選的驗證和字符限制 |
| 多行文本 | 多行文本字段，支持豐富文本和格式選項 |
| 單選 | 下拉選單，單一選項選擇和自定義選項 |
| 多選 | 下拉選單，多個選項選擇和自定義選項 |
| 位置 | 經度和緯度，帶地圖視圖和地址查找 |
| 復選框 | 真/假切換，具有可選的默認值和自定義標籤 |
| 國家 | 兩個字母的 ISO 國家代碼選擇器，帶國家名稱 |
| 貨幣 | 數字字段，帶貨幣符號和匯率支持（例如，USD200，200USD，KHR4000） |
| 日期 | 日期選擇器，具有可選的時間組件和時區支持 |
| 電話 | 支持 E.164、國際格式或國碼 + 號碼的電話號碼字段，帶驗證 |
| 電子郵件 | 電子郵件字段，具有格式驗證和域名驗證 |
| 星級評價 | 1-5 的數字評分，帶視覺星星和自定義標籤 |
| 百分比 | 帶百分比符號（0-100）和小數精度的數字字段 |
| URL | 網址字段，具有格式驗證和預覽 |
| 數字 | 數字字段，具有可選的小數位和單位顯示 |
| 公式 | 基於其他字段的計算字段，支持數學運算 |
| 唯一 ID | 自動生成的標識符，具有自定義前綴和格式 |
| 參考 | 連結到另一條記錄，具有搜索和過濾選項 |
| 查找 | 動態字段，從引用的記錄中提取數據，具有映射選項 |
| 時間持續 | 以小時和分鐘為單位的時間跨度，具有自定義顯示格式 |

### 更新記錄
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 列表：列表 ID
- 記錄 ID：要更新的記錄的 ID
- 記錄名稱：記錄的名稱
- 默認字段：默認值
- 單一日期：截止日期
- 日期範圍：開始和截止日期
- 被指派者：多選用戶
- 標籤：多選標籤
- 描述：文本字段，支持 HTML
- 清單：清單標題的數組

**支持的自定義字段：**（與“創建記錄”相同）

### 搜索記錄
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 列表：列表 ID
- 搜索/查詢：帶有高級過濾選項的搜索文本

### 獲取記錄
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 列表：列表 ID
- 記錄 ID：要檢索的記錄 ID，及所有相關數據

### 邀請用戶
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 電子郵件地址：用戶的電子郵件地址，帶驗證
- 角色 ID：角色 ID（默認或自定義），帶權限

### 列出自定義字段
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 限制：返回的最大字段數量，帶分頁

### 列出列表
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 限制：返回的最大列表數量，帶排序選項

### 列出所有項目
- 公司 ID：多選公司 ID 或別名
- 限制：返回的最大項目數量，帶過濾

### 列出標籤
- 公司 ID：公司 ID 或別名
- 項目 ID：項目 ID 或別名
- 限制：返回的最大標籤數量，帶使用統計

### 執行 GraphQL
- 標頭：用於身份驗證和自定義標頭的鍵值對數組
- 方法：GET 或 POST，帶請求選項
- 查詢：GraphQL 查詢或變更，支持變量
- 操作名稱：用於跟踪的操作名稱
- 變量數據來源：原始、表單或集合，帶類型檢查
- 變量：GraphQL 變量內容，帶驗證
