Los tags son una forma útil de organizar su trabajo. Puede usarlos para mostrar diferentes tipos de tareas, urgencia, o cualquier categoría que se adapte a su proyecto. Cada tag puede tener un código de color para una fácil identificación y para agregar algo de color a su espacio de trabajo.

<youtube url="https://www.youtube.com/watch?v=nlyY3cUUpkQ" />


## Usando Tags

- **Agregando Tags**: Para crear un tag, considere los aspectos de sus tareas que necesitan categorización—ya sea la naturaleza de la tarea, como 'bug', 'back-end', o 'UI', o su prioridad. Una vez que haya identificado sus categorías, puede crear fácilmente tags y asignarles colores únicos para un reconocimiento inmediato.
- **Filtrando con Tags**: Las capacidades de filtrado de Blue hacen que gestionar su carga de trabajo sea más intuitivo. Puede filtrar tags usando el ícono desplegable de Filtro o directamente dentro de la vista de tablero haciendo clic en el tag deseado.

Etiquetado Entre Proyectos <br/>
Para vistas de toda la organización como Dashboards y mapas, el etiquetado se vuelve especialmente importante porque Blue automáticamente fusiona tags del mismo nombre. Así que solo necesita filtrar por el tag "Prioridad 1", lo cual mostrará únicamente registros de todos los proyectos con ese tag.
