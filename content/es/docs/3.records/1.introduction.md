## Descripción General

Los registros en Blue cumplen dos propósitos clave:

1. **Seguimiento de Elementos de Trabajo**: Puede usar los registros como listas de tareas dinámicas para rastrear el progreso de tareas, proyectos, tratos de ventas, etc. Asigne fechas de vencimiento, etiquetas, listas de verificación y campos personalizados para capturar toda la información relevante.
2. **Organización de Información**: Los registros le permiten estructurar información usando listas como categorías. Por ejemplo, puede tener una lista de "Prospectos Abiertos" y "Tratos Cerrados" para organizar su pipeline de ventas.

## Características Principales

Estas son algunas de las características principales de los registros en Blue:
- [Listas](/docs/records/lists): Organice registros en diferentes listas para categorizar información. Por ejemplo, tenga listas de "Por Hacer", "En Progreso" y "Completado".
- [Campos Personalizados](/docs/custom-fields/introduction): Agregue campos personalizados como menús desplegables, números y fechas para capturar información especializada. Por ejemplo, agregue un campo numérico de "Puntuación de Prospecto".
- [Etiquetas](/docs/records/tags): Use etiquetas coloridas como "Urgente" o "Error" para destacar registros prioritarios.
- [Comentarios](/docs/records/comments): Colabore con su equipo comentando directamente en los registros.
- [Archivos Adjuntos](/docs/projects/file-management): Agregue archivos, imágenes, videos directamente a los registros para fácil acceso.
- [Automatizaciones](/docs/automations/introduction): Active automatizaciones cuando los registros se crean, editan o eliminan. Por ejemplo, envíe una alerta por email cuando se genere un registro de alta prioridad.
- **Colores**: Establezca colores para sus registros para identificar la urgencia de los registros o para planificación de contenido.

## Casos de Uso

Estos son algunos ejemplos de cómo puede usar los registros en su organización:

### Gestión de Tareas y Proyectos

Use registros para rastrear el progreso de tareas y proyectos con asignados, fechas de vencimiento, archivos adjuntos, etc. Agrúpelos en listas como "Backlog", "En Progreso" y "En Vivo" para visualizar el flujo de trabajo.

### Pipeline de Ventas

Gestione su pipeline de ventas teniendo registros para cada prospecto y oportunidad. Rastree el valor del trato, puntuación del prospecto y próximos pasos en campos personalizados. Organice registros en listas como "Tratos Abiertos" y "Cerrados Ganados" para analizar su embudo.

### Tickets de Soporte

Los registros pueden actuar como tickets de soporte para rastrear problemas de clientes. Los agentes pueden ser asignados a tickets, hacer preguntas aclaratorias en comentarios y actualizar el estado en un campo personalizado. Agrupe tickets por tipo usando etiquetas como "Error", "Pregunta", etc.

### Reclutamiento

En su proceso de contratación, tenga un registro para cada aplicación de candidato con información clave en campos personalizados. Muévalos a través de listas como "Aplicado", "Entrevista Telefónica" y "Ronda Final" para rastrear la progresión. Use automatización de email para enviar actualizaciones al candidato cuando el registro se mueva a otra lista.

En resumen, los registros le dan una forma flexible de construir un flujo de trabajo personalizado que se ajuste a su proceso de negocio. Con características como listas, campos personalizados y automatizaciones, puede construir algunos sistemas poderosos sobre los registros en Blue.
