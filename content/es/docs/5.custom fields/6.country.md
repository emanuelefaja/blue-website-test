El campo personalizado País le permite asociar registros en Blue con países específicos. Proporciona una lista desplegable que contiene todos los países reconocidos en el mundo, permitiéndole seleccionar el país relevante para cada registro.

![Country 1](/docs/Country_1.png "Country 1")

El campo personalizado País mostrará un menú desplegable de todos los países reconocidos

El menú desplegable muestra el nombre del país y el ícono de la bandera nacional, proporcionando una identificación visual clara. Puede seleccionar múltiples países por registro si es necesario.

![Country 2](/docs/Country_2.png "Country 2")

Los usuarios pueden seleccionar múltiples países al mismo tiempo

Para ubicar rápidamente un país, comience a escribir el nombre - el menú desplegable se filtrará instantáneamente a los países coincidentes en una interfaz de búsqueda mientras escribe. Esto hace que encontrar incluso nombres de países menos conocidos sea rápido y fácil.

![Country 3](/docs/Country_3.png "Country 3")

Busque instantáneamente cualquier país

La lista completa está organizada alfabéticamente para facilitar el escaneo. Una vez seleccionado, el país o países elegidos se mostrarán usando los íconos de las banderas nacionales para referencia visual continua.

Los usos potenciales para el campo País incluyen:

- Capturar ubicaciones de clientes, socios o proveedores
- Asociar registros con regiones geográficas relacionadas
- Filtrar datos por país para análisis y reportes
- Rastrear destinos de envío internacional

Con soporte robusto para todos los países globales y una interfaz de búsqueda refinada, el campo desplegable País enriquece los registros con datos geográficos relevantes.
