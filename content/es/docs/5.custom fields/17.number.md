---
title: Número
description: 
---

El campo personalizado Número permite capturar valores de datos numéricos dentro de límites mínimos y máximos configurados.

![Number](/docs/Number.png "Number")

Los [Administradores de Proyecto](/es/docs/user-management/roles/project-administrator) pueden definir el rango permitido al configurar el campo personalizado Número. El sistema validará las entradas para asegurar que estén dentro de los valores mínimos y máximos permitidos.
