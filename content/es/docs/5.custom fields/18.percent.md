El campo personalizado de Porcentaje permite capturar y calcular valores de porcentaje dentro de los registros de Blue. Esto habilita el seguimiento del progreso, la medición del rendimiento y la visualización de datos basada en porcentajes.

![Percent](/docs/Percent.png "Percent")

El campo personalizado de Porcentaje viene con validación robusta para respaldar datos precisos. Al introducir valores, verifica que el número esté dentro del rango estándar de 0-100% esperado para porcentajes. Esto previene que se introduzcan datos inválidos.

Una característica clave es la configuración flexible de la precisión decimal. Puede configurar el campo para capturar porcentajes con tantos lugares decimales como se requiera según sus necesidades. Este nivel de personalización permite el seguimiento de datos basados en porcentajes en cualquier nivel de granularidad que requiera su caso de uso.

Para simplificar los cálculos, el campo de Porcentaje se integra directamente con [campos personalizados de Fórmula](/es/docs/custom-fields/formula). Esto hace fácil automatizar barras de progreso, ratios, tasas y otras métricas comunes basadas en porcentajes. Los cálculos ocurren detrás de escena, mostrando el valor de porcentaje de salida.

Al exportar datos a través de integraciones de [CSV](/es/docs/integrations/csv-export) o [API](/es/docs/integrations/api), el campo mantiene su visualización de porcentaje formateada, incluyendo cualquier decimal. Esto asegura que los datos de Porcentaje se exporten con precisión mientras se mantiene la precisión y formato configurados por el usuario.
