El campo personalizado de Casilla de verificación proporciona una casilla que se puede utilizar para una variedad de propósitos, incluyendo aprobaciones o [para activar una automatización](/es/docs/automations/introduction)

![Checkbox 2](/docs/Checkbox_1.png "Checkbox 1")

Uso de campos personalizados de Casilla de verificación para aprobaciones

Las aprobaciones son un caso de uso específicamente bueno para el campo personalizado de Casilla de verificación debido a las capacidades de auditoría de Blue. Cada vez que se marca o desmarca la casilla, Blue registra la acción de manera permanente:

![Checkbox 2](/docs/Checkbox_2.png "Checkbox 2")

Blue registra la persona, la acción y el tiempo para cada cambio.
