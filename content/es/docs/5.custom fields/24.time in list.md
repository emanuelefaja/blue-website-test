El campo personalizado Tiempo en Lista rastrea automáticamente cuánto tiempo ha estado un registro en cada lista, proporcionando información detallada sobre la progresión del flujo de trabajo y los cuellos de botella.

El campo Tiempo en Lista es extremadamente valioso para analizar la eficiencia del flujo de trabajo e identificar mejoras del proceso. Algunos ejemplos incluyen:

- Monitorear cuánto tiempo los tickets de soporte pasan en cada estado (Nuevo, En Progreso, Esperando Cliente, Resuelto)
- Rastrear la progresión del pipeline de ventas para identificar dónde se atascan las oportunidades
- Analizar las etapas del flujo de trabajo de manufactura para optimizar los cronogramas de producción
- Medir los procesos de aprobación de contenido desde borrador hasta publicado
- Identificar cuellos de botella en flujos de trabajo de reclutamiento desde aplicación hasta contratación
- Rastrear fases de proyecto para entender dónde ocurren comúnmente los retrasos
- Monitorear etapas de entrega de servicio para mejorar la experiencia del cliente

El campo Tiempo en Lista proporciona seguimiento automático en tiempo real sin intervención manual. Calcula el tiempo gastado en cada lista tanto histórica como actualmente, mostrando desgloses porcentuales de dónde los registros pasan más tiempo. Esto permite optimización de procesos basada en datos, ayuda a identificar ineficiencias y proporciona visibilidad clara del rendimiento del flujo de trabajo.

## Cómo Funciona

El seguimiento de Tiempo en Lista comienza automáticamente cuando agregas el campo personalizado Tiempo en Lista a tu proyecto. El sistema:

1. **Rastrea Tiempos de Entrada y Salida**: Registra marcas de tiempo precisas cuando los elementos entran y salen de cada lista
2. **Calcula Duración**: Calcula automáticamente el tiempo gastado usando la diferencia entre tiempos de entrada y salida
3. **Maneja Estado Actual**: Para elementos actualmente en una lista, calcula el tiempo desde la entrada hasta ahora
4. **Preserva Historial**: Mantiene seguimiento completo de tiempo incluso si las listas son renombradas o eliminadas
5. **Muestra Desglose Porcentual**: Muestra qué porción del tiempo total se gastó en cada lista

## Opciones de Visualización

Los datos de Tiempo en Lista aparecen en múltiples ubicaciones:

- **Tarjetas de Registro**: Muestra tiempo de lista actual y tiempo total del proyecto con iconos intuitivos
- **Vista de Detalle de Registro**: Muestra desglose comprensivo con barras de progreso para cada lista
- **Vista de Base de Datos**: Los datos de tiempo pueden mostrarse en formato de tabla para análisis
- **Integración de Campo Personalizado**: Funciona junto con campos personalizados TIME_DURATION para seguimiento avanzado

## Configuración

El campo Tiempo en Lista no requiere configuración - rastrea automáticamente todos los movimientos de lista. Sin embargo, puedes controlar las opciones de visualización:

- **Mostrar Tiempo en Lista Actual**: Alternar visualización del tiempo gastado en la lista actual en tarjetas de registro
- **Mostrar Tiempo en Proyecto**: Alternar visualización del tiempo total a través de todas las listas en el proyecto
- **Visibilidad de Campo Personalizado**: Controlar si el seguimiento de tiempo aparece en detalles de registro

## Beneficios Clave

- **Configuración Cero Requerida**: Comienza automáticamente el seguimiento cuando se agrega el campo
- **Historial Completo**: Mantiene seguimiento incluso a través de cambios y eliminaciones de listas
- **Actualizaciones en Tiempo Real**: Los cálculos de tiempo se actualizan continuamente para elementos actuales
- **Indicadores de Progreso Visual**: Las barras de progreso muestran distribución relativa de tiempo a través de listas
- **Optimización de Procesos**: Identifica dónde los flujos de trabajo se ralentizan o se atascan
- **Métricas de Rendimiento**: Genera información para productividad del equipo y eficiencia del proceso
