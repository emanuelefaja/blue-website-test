---
title: List Automations
description: How to list automations using the Blue API.
---

To list automations in a specific project, you can make the following query:

```graphql
query AutomationList {
  automationList {
    items {
      id
      isActive
      updatedAt
      createdAt
      trigger {
        id
        type
        color
      }
      actions {
        id
        type
        color
      }
    }
  }
}
```

Ensure to pass the `companyId` and `projectId` in the query.
