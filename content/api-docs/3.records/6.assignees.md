---
title: Assign Record 
description: Learn how to assign records to users using the Blue API.
---

To assign a record to a user, you can make the following mutation:

```graphql
mutation SetTodoAssignees {
  setTodoAssignees(
    input: {
      todoId: "todo-id"
      assigneeIds: ["assignee-id"]
    }
  ) {
    success
  }
}
```
