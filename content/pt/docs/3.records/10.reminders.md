Os lembretes no Blue podem enviar um e-mail ou notificação push móvel para qualquer registro em um horário predeterminado. Seja para responder a um e-mail urgente, enviar um relatório ou retornar uma ligação para um cliente, você pode agendar lembretes para que suas responsabilidades mais importantes sejam sempre cumpridas no prazo. Nós cuidamos do envio dos lembretes - você só precisa focar em completar o trabalho.

## Configurando um Lembrete

Para configurar um lembrete em um registro:
1. Abra o registro
2. Clique no ícone do relógio no canto superior direito
3. Selecione a data e horário desejados para o lembrete

![Lembrete 1](/docs/Reminder_1.png "Lembrete 1")

Ou por padrão você pode selecionar:

- Mais tarde hoje
- Amanhã
- Mais tarde esta semana
- Este fim de semana
- Próxima semana

![Lembrete 2](/docs/Reminder_2.png "Lembrete 2")

Uma vez que você configure um lembrete, verá a data e hora exibidas diretamente no registro.

## Configurações de Notificação

Quando a data e hora do lembrete ocorrer, você receberá uma notificação por:
- E-mail (se habilitado nas suas configurações de notificação)
- Notificação push nos aplicativos web/móvel do Blue (se habilitado)

A notificação o alertará para revisitar o registro.

Nas suas configurações pessoais de notificação, você pode ativar/desativar especificamente as notificações por e-mail e push para lembretes. Isso permite escolher como você quer ser notificado para lembretes.

Por padrão, ambas as opções estão habilitadas quando os lembretes são configurados pela primeira vez.

![Lembrete 3](/docs/Reminder_3.png "Lembrete 3")

## Editando ou Excluindo Lembretes

Para editar ou remover um lembrete:
1. Passe o mouse sobre a data no registro
2. Escolha editar ou excluir o lembrete

Editar permite adiar o lembrete para uma data posterior. Excluir removerá completamente o lembrete.

![Lembrete 4](/docs/Reminder_4.png "Lembrete 4")
