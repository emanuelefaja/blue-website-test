Campos personalizados de referência permitem vincular registros de diferentes projetos. Eles fornecem um menu suspenso que permite selecionar um registro de outro projeto.
Campos personalizados de referência funcionam junto com [campos personalizados de consulta](/pt/docs/custom-fields/lookup) para trazer informações de um projeto para outro. Isso é extremamente poderoso porque desbloqueia muitos casos de uso dentro do Blue.

A empresa ACME usa campos personalizados de Referência e Consulta do Blue para criar um ecossistema de dados interconectado em projetos separados de Clientes, Vendas e Estoque. Registros de clientes no projeto Clientes são vinculados via Referência a transações de vendas no projeto Vendas, permitindo que campos personalizados de Consulta tragam detalhes de clientes associados como número de telefone e status da conta diretamente para cada registro de venda; além disso, os itens de estoque vendidos podem ser exibidos no registro de venda através de um campo de Consulta referenciando dados de Quantidade Vendida do projeto Estoque; finalmente, retiradas de estoque são conectadas a vendas relevantes via um campo de Referência no Estoque apontando de volta para os registros de Vendas, fornecendo visibilidade completa sobre qual venda acionou a remoção do estoque. Ao aproveitar Referências e Consultas, a ACME vincula clientes, vendas e estoque em uma visão integrada de 360 graus entre projetos.

## Configuração

Ao criar um campo de Referência, o [Administrador do Projeto](/pt/docs/user-management/roles/project-administrator) seleciona o projeto específico que fornecerá a lista de registros de referência.

Em seguida, eles configuram se o campo permitirá:
- Seleção única: Escolher um registro de referência
- Seleção múltipla: Escolher múltiplos registros de referência

Finalmente, um filtro pode ser definido para permitir que usuários selecionem apenas registros que correspondam ao filtro.

## Rastrear registro referenciado.

Às vezes, quando você tem registros referenciados em múltiplos projetos, você quer ver instantaneamente onde eles foram referenciados, tudo em um só lugar. Você pode fazer isso configurando um campo personalizado 'Referenciado Por'.

Isso permite que você veja onde seu registro foi referenciado em diferentes projetos para que você tenha um rastro de onde ele foi referenciado. Com apenas um clique, você será levado para esse projeto específico instantaneamente.

![Reference 3](/docs/Reference_3.png "Reference 3")

O campo referenciado por será automaticamente preenchido com todos os registros para mostrar onde foi referenciado

![Reference 4](/docs/Reference_4.png "Reference 4")

Esta funcionalidade é particularmente útil quando você quer ver sua lista de contatos de clientes sendo referenciada em múltiplos projetos e fornecendo uma maneira rápida e fácil de rastrear o uso de registros entre projetos.

Para remover este referenciado por do seu registro:

1. Vá para "Gerenciar Campos"
2. Clique na lixeira vermelha no campo "Referenciado Por" para deletar.
