O campo personalizado País permite que você associe registros no Blue com países específicos. Ele fornece uma lista suspensa contendo todos os países reconhecidos no mundo, permitindo que você selecione o país relevante para cada registro.

![Country 1](/docs/Country_1.png "Country 1")

O campo personalizado País mostra uma lista suspensa de todos os países reconhecidos

O menu suspenso exibe o nome do país e o ícone da bandeira nacional, fornecendo identificação visual clara. Você pode selecionar múltiplos países por registro, se necessário.

![Country 2](/docs/Country_2.png "Country 2")

Os usuários podem selecionar múltiplos países ao mesmo tempo

Para localizar rapidamente um país, comece digitando o nome - a lista suspensa filtrará instantaneamente os países correspondentes em uma interface de busca em tempo real. Isso torna a localização de nomes de países, mesmo os menos conhecidos, rápida e fácil.

![Country 3](/docs/Country_3.png "Country 3")

Busque instantaneamente por qualquer país

A lista completa está organizada alfabeticamente para facilitar a navegação. Uma vez selecionado, o país ou países escolhidos serão exibidos usando os ícones das bandeiras nacionais para referência visual contínua.

Os usos potenciais para o campo País incluem:

- Capturar localizações de clientes, parceiros ou fornecedores
- Associar registros com regiões geográficas relacionadas
- Filtrar dados por país para análise e relatórios
- Rastrear destinos de envio internacional

Com suporte robusto para todos os países globais e uma interface de busca refinada, o campo suspenso País enriquece os registros com dados geográficos relevantes.
