O campo personalizado Caixa de Seleção fornece uma caixa de seleção que pode ser usada para diversos propósitos, incluindo aprovações ou [para acionar uma automação](/pt/docs/automations/introduction)

![Checkbox 2](/docs/Checkbox_1.png "Checkbox 1")

Usando campos personalizados de Caixa de Seleção para aprovações

As aprovações são um caso de uso especialmente bom para o campo personalizado Caixa de Seleção devido às capacidades de auditoria do Blue. A cada vez que a caixa de seleção é marcada ou desmarcada, o Blue registra a ação de forma permanente:

![Checkbox 2](/docs/Checkbox_2.png "Checkbox 2")

O Blue registra a pessoa, ação e horário para cada alteração.
