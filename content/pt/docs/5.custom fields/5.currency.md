Armazene informações financeiras no campo de moeda

Ao adicionar valores de moeda, o campo formatará o número de acordo com as convenções locais e exibirá o símbolo de moeda relevante. Por exemplo, $1,000.50 para Dólares Americanos ou €1.500,25 para Euros.

O campo de moeda valida as entradas para garantir que apenas valores de moeda válidos sejam capturados. Uma vez adicionado a um projeto, o campo armazenará quantias no tipo de moeda singular designado. Ele não fornece conversão automática de moeda entre múltiplas denominações.

Este campo oferece uma experiência otimizada para rastrear dados monetários associados aos registros. A formatação automatizada remove a complexidade de inserir manualmente valores de moeda em formatos locais, enquanto a validação protege a integridade dos dados.

Os usos potenciais para o campo de moeda incluem:

- Capturar valores de transações
- Armazenar orçamentos de projetos
- Rastrear totais de faturas
- Registrar valores de pagamentos
- Calcular receita

Com suporte robusto para formatos de moeda globais e entrada de dados simplificada, o campo personalizado de Moeda permite que você centralize informações financeiras importantes dentro de seus registros no Blue.
