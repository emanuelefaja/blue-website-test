O campo personalizado de Porcentagem permite capturar e calcular valores percentuais dentro dos registros do Blue. Isso habilita rastreamento de progresso, medição de performance e visualização de dados baseada em porcentagens.

![Percent](/docs/Percent.png "Percent")

O campo personalizado de Porcentagem vem com validação robusta para suportar dados precisos. Ao inserir valores, verifica se o número está dentro da faixa padrão de 0-100% esperada para porcentagens. Isso previne que dados inválidos sejam introduzidos.

Um recurso principal é a configuração flexível da precisão decimal. Você pode definir o campo para capturar porcentagens com quantas casas decimais forem necessárias baseado em suas necessidades. Este nível de personalização permite rastreamento de dados baseado em porcentagens em qualquer nível de granularidade que seu caso de uso requeira.

Para simplificar cálculos, o campo Porcentagem integra diretamente com [campos personalizados de Fórmula](/pt/docs/custom-fields/formula). Isso torna fácil automatizar barras de progresso, proporções, taxas e outras métricas comuns baseadas em porcentagem. Os cálculos acontecem nos bastidores, exibindo o valor percentual de saída.

Ao exportar dados via integrações de [CSV](/pt/docs/integrations/csv-export) ou [API](/pt/docs/integrations/api), o campo mantém sua exibição percentual formatada, incluindo quaisquer decimais. Isso assegura que os dados de Porcentagem sejam exportados com precisão enquanto mantém a precisão e formatação configuradas pelo usuário.
