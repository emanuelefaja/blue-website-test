title: Número
description: 
---

O campo personalizado Número permite capturar valores de dados numéricos dentro de limites mínimos e máximos configurados.

![Number](/docs/Number.png "Number")

[Administradores de Projeto](/pt/docs/user-management/roles/project-administrator) podem definir o intervalo permitido ao configurar o campo personalizado Número. O sistema validará as entradas para garantir que estejam dentro dos valores mínimos e máximos permitidos.
