---
title: Arquivar projetos
description: Congele projetos uma vez concluídos para conformidade.
---

Em muitas organizações, reter dados de projetos concluídos é crucial para fins de auditoria ou para revisitar o projeto no futuro. No entanto, acumular centenas de projetos concluídos pode sobrecarregar a interface do Blue, dificultando a navegação e o gerenciamento de projetos atuais.

É por isso que o Blue oferece a capacidade para [Administradores de Projeto](/docs/user-management/roles/project-administrator) arquivarem projetos uma vez que estejam completos ou não sejam mais necessários.

![Archiving Project 1](/docs/ArchivingProject_1.png "Archiving Project 1")

Arquive facilmente qualquer projeto

Quando um projeto é arquivado, várias coisas acontecem.

- **Realocação para 'Projetos Arquivados'**: O projeto é movido da área principal 'Projetos' na barra lateral para uma seção designada 'Projetos Arquivados'. Isso ajuda a manter uma interface principal limpa e organizada.
- **Status Arquivado Visível**: Na barra superior, aparece um rótulo 'Arquivado' ao lado do nome do projeto, indicando claramente seu status.
- **Acesso Somente Leitura**: Projetos arquivados tornam-se somente leitura. Eles são essencialmente 'congelados no tempo', significando que todos os usuários podem visualizar o projeto e seu conteúdo, mas nenhuma modificação ou adição pode ser feita. Isso garante a integridade e imutabilidade dos dados do projeto como estavam no momento do arquivamento.
- **Mudanças no Acesso à API**: Embora o projeto ainda possa ser acessado através da API do Blue para operações de leitura, quaisquer operações de escrita (tentativas de modificar ou adicionar dados) não serão bem-sucedidas. Isso mantém o status somente leitura do projeto mesmo através de interações da API.
